# JackSsh Project Instructions

JackSsh is a native iOS application.

## Stack

- Swift
- SwiftUI
- MVVM
- Swift Package Manager
- Swift Testing
- SwiftData
- Keychain
- LocalAuthentication

## Modules

- JackSsh
- Presentation
- Domain
- Data
- DesignSystem
- Shared

## Mandatory workflow

1. Inspect the repository before modifying files.
2. Preserve existing changes.
3. Work by feature.
4. Run real tests and builds.
5. Never claim success without command output.
6. Do not invent APIs or infrastructure.
7. Keep the app target thin.
